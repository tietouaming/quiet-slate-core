## 摘要
在既有 `50篇顶刊文献审查` 基础上，本报告再补充 20 篇与“镁合金孪晶-晶体塑性-腐蚀耦合相场”直接相关的高水平期刊论文。
写作顺序遵循“先系统回顾相关工作，再阐述本工作”的论文规范；并明确推理过程中的前提条件、适用边界与不确定性来源。

## 1. 研究问题与前提
目标问题：在微观相场与晶体塑性可解释性的前提下，构建可用于宏观腐蚀预测与反向设计的多尺度模型。

本报告采用的前提条件如下：
1. 当前代码框架以小应变为主，孪晶通过序参量-本征应变耦合进入力学。
2. 腐蚀采用相场界面演化，浓度场采用化学势一致的守恒扩散离散。
3. 机械耦合默认走 mobility-only 路径；能量门控作为数值一致性判据，不等同严格热力学证明。
4. 文献纳入优先级：`Mg + twinning/CP + corrosion/phase-field`，其次为直接方法支撑论文。

## 3. 从相关工作到本工作的推理过程
### 3.1 推理链条
1. 早期腐蚀实证文献（AZ91D/AZ31/ZE41/AM50）给出可观测机理边界。
2. 孪晶与晶体塑性相场文献给出 η-界面-本征应变的可计算表达。
3. SCC/电化学-力学相场文献给出 corrosion-chemistry-mechanics 三场耦合的组织框架。
4. 由此确定本工作采用：Allen-Cahn(phi) + mu-扩散(c) + CP/孪晶(eta,epsp) + 准静力学。
5. 在无实验大数据前提下，采用“高保真物理仿真生成训练集 -> 宏观 surrogate 学习 -> 物理门控回退”的闭环。

### 3.2 关键前提条件
1. 尺度前提：当前是 2D 小应变框架，适于机制识别与参数筛选，不直接等价全三维工程构件。
2. 数据前提：训练数据主要来自数值仿真，宏观模型正确性需通过物理一致性指标而非单纯统计误差判定。
3. 门控前提：surrogate 只能在残差/守恒/能量判据通过时替代物理步。

### 3.3 评估指标（无大规模宏观真值时）
1. PDE 残差无量纲指标：`R_phi, R_c, R_eta, R_mech`。
2. 守恒一致性：浓度均值投影误差与边界通量一致性。
3. 结构响应指标：腐蚀前沿推进率、固相体积分数、峰值/均值应力演化。
4. 跨尺度一致性：surrogate 与锚定物理步在关键物理量上的漂移上界。

## 4. 本工作的定位与贡献（在70篇文献背景下）
1. 理论层：将 Mg 腐蚀相场、CP-孪晶和力学平衡放入统一可计算框架，并保持主要变量口径一致。
2. 数值层：使用非均匀网格二阶算子 + FV 守恒扩散 + Krylov 力学求解，兼顾稳定性与可扩展性。
3. 机器学习层：采用“预测-校正-门控-回退”的安全加速策略，优先保证正确性。
4. 工程层：保留电场/氢脆扩展接口，支持后续多物理增强。

## 5. 仍需继续完善的点
1. 有限应变孪晶运动学与 3D 多变体一致性。
2. 更严格热力学闭合（由 mobility-only 向全变分耦合过渡）。
3. 宏观 surrogate 的不确定性量化与长时外推误差界。
