# 目录结构说明

本文说明 `d:\zip_pojie` 当前的项目分区，便于快速定位代码、数据与工具脚本。

## 1. 顶层目录

- `mg_coupled_pf/`：核心仿真代码（相场-晶塑-孪晶-腐蚀耦合）
- `configs/`：算例与模型配置（YAML）
- `scripts/`：运行、训练、基准、导出等入口脚本
- `tests/`：自动化测试
- `artifacts/`：仿真、训练、报告输出
- `docs/`：中文手册与技术说明
- `data/`：原始文档、提取文本、Office 导出中间产物
- `tools/`：与主仿真解耦的独立工具

## 2. data 子目录

- `data/raw_documents/`
  - 原始研究文档（`pptx/docx/pdf`）
- `data/extracted_text/`
  - 文档抽取文本与中间 jsonl
- `data/office_exports/`
  - Word/PPT 导出资源（html/mht/pdf/图片目录）
- `data/cache_archive/`
  - 缓存目录归档（替代直接删除）

## 3. tools 子目录

- `tools/standalone/caoliu_scraper.py`
- `tools/standalone/zip.py`

说明：这两个脚本不属于核心耦合仿真流程，已从根目录分离。

## 4. 本次整理后的使用建议

- 运行仿真：优先使用 `scripts/` 下脚本与 `configs/` 下配置。
- 处理原始文档：统一从 `data/raw_documents/` 读取。
- 公式提取命令已更新为新路径（见 `docs/中文操作手册.md`）。
- 独立抓取/压缩工具统一在 `tools/standalone/` 维护。
